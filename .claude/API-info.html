FileList offers a JSON/XML API that will allow you to get information from our website.
The API is available at the following URL: https://filelist.io/api.php


Quote:
General Info


All calls to the API have to be sent as a GET request;
All the arguments listed below are set as parameters in the call URL;
Every API call requires authentication with Username and Passkey (can be found here);
Every user is limited to 150 API calls per hour;
Red highlighted parameters are required!

Quote:
Authentication


Parameters:
   username
   passkey
Example: https://filelist.io/api.php?username=ToXiC&passkey=1234567890

OR you can set the "Authorization" header:
Value:
  Basic username:passkey - "username:passkey" must be encoded in base64
Example: taking "ToXiC:1234567890" as "username:passkey", you must set the value as: Basic VG9YaUM6MTIzNDU2Nzg5MA==

Quote:
Parameters / Endpoints


Parameters:
   action
Valid values: search-torrents, latest-torrents
Example: https://filelist.io/api.php?username=ToXiC&passkey=1234567890&action=search-torrents



Additional parameters for action=search-torrents
   type                Valid values: imdb, name
   query              If you choose imdb as type, it is accepted in two forms: tt4719744 or 4719744;
   name              Accepted as an optional parameter if type=imdb. Also searches in the name field.
   category          Valid values: IDs from categories, multiple values ​​separated by a comma are accepted.
   moderated       Valid values: 0,1
   internal            Valid values: 0,1
   freeleech          Valid values: 0,1
   doubleup          Valid values: 0,1
   output              Valid values: json, rss. Default is JSON.
   season              Valid values integers
   episode             Valid values integers

Additional parameters for action=latest-torrents
   limit          Maximum number of torrents displayed in the request. Can be 1-100. Default value: 100
   imdb         Accepted as: tt4719744 or 4719744
   category    Valid values: IDs from categories, multiple values ​​separated by a comma are accepted.
   output       Valid values: json, rss. Default is JSON.


Example:
   https://filelist.io/api.php?username=ToXiC&passkey=1234567890&action=search-torrents&type=name&query=Gemini
   https://filelist.io/api.php?username=ToXiC&passkey=1234567890&action=search-torrents&type=imdb&query=tt4719744&category=4,19
   https://filelist.io/api.php?username=ToXiC&passkey=1234567890&action=latest-torrents
   https://filelist.io/api.php?username=ToXiC&passkey=1234567890&action=latest-torrents&output=rss
